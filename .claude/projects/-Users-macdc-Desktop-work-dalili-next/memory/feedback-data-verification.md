RÈGLE ABSOLUE : verify every figure on its official source before writing it. Never write a price, amount, delay, or quota from memory.

**Why:** Figures (APL amounts, CVEC, tuition fees, SMIC, CSS plafond, visa delays) are revised every year. Writing outdated figures destroys trust and harms students.

**How to apply:** Before writing any number in a blog article, run a WebSearch or WebFetch on the relevant official source:

| Data type | Official source |
|---|---|
| APL/CAF amounts | caf.fr (simulateur officiel) |
| CVEC 2026 | messervices.etudiant.gouv.fr |
| Tuition fees (hors UE) | site officiel de chaque université + campusfrance.org/fr/les-frais-de-scolarite |
| SMIC horaire | travail-emploi.gouv.fr |
| Student work hours limit | service-public.fr |
| Visa delays by country | france-visas.gouv.fr |
| Sécu / CPAM / CSS plafond | ameli.fr |
| TCF fees | france-education-international.fr |
| Bourse Eiffel amounts | campusfrance.org |
| OFII | ofii.fr |
| Campus France procedures | campusfrance.org |
| Titre de séjour (ANEF) | administration-etrangers-en-france.interieur.gouv.fr |

If the official source gives a range → use that exact range.
If the info cannot be found → write "Vérifier sur [source] car les montants sont révisés chaque année" — never invent a figure.

**Footer to add on every article with figures:**
> ⚠️ Ces informations sont vérifiées en juin 2026. Certains montants (CAF, CVEC, frais universitaires) sont révisés chaque année. Vérifiez toujours sur les sites officiels avant de prendre une décision.
